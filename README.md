# Claude Code 和 Codex 中转网站

收集国内可用的 Claude Code 和 Codex 中转服务，帮助用户更方便地使用 AI 编程工具。

> ⚠️ 使用中转站前请仔细阅读以下提示：

1. **模型真实性**：部分中转站可能存在模型混用问题（用低价模型替代高价模型），建议选择信誉良好的站点
2. **稳定性**：中转站可能因网络攻击、服务器问题等导致服务中断，重要工作请本地备份
3. **价格波动**：中转站价格可能随官方定价调整，请以实际为准
4. **隐私安全**：避免在中转站输入敏感代码或机密信息
5. **退款政策**：购买前确认退款政策，优先选择支持退款的站点
